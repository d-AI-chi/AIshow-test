## 次のステップ

### 1. GitHubでリポジトリを作成済みの場合

リポジトリ名を教えてください。以下のコマンドを実行します：

```bash
cd "/Users/daichishimizu/Documents/01. Private/AIshow-test"
git remote add origin https://github.com/d-AI-chi/リポジトリ名.git
git push -u origin main
```

### 2. まだリポジトリを作成していない場合

1. https://github.com/new にアクセス
2. Repository name を入力（例: `AIshow-test` または `aishow-diagnosis`）
3. Public または Private を選択
4. **重要**: チェックボックスはすべて外す
5. 「Create repository」をクリック
6. 作成したリポジトリ名を教えてください

## よく使うリポジトリ名の例

- `AIshow-test`
- `aishow-diagnosis`
- `aishow-matching`
- `ai-show-test`

リポジトリ名を教えていただければ、すぐに接続コマンドを実行します！
