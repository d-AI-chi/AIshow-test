# サービスサイトのリンク確認方法

## VercelサイトのURLを確認する方法

### 方法1: Vercelダッシュボードから確認（推奨）

1. [Vercelダッシュボード](https://vercel.com/dashboard)にアクセス
2. プロジェクト一覧から「**a-ishow-test**」をクリック
3. プロジェクトページの上部に表示されるURLを確認
   - 通常は `https://プロジェクト名.vercel.app` の形式

### 方法2: Domainsタブから確認

1. Vercelダッシュボードでプロジェクトを開く
2. 「**Settings**」タブをクリック
3. 左側メニューから「**Domains**」をクリック
4. 表示されているドメインを確認

## 一般的なURL形式

プロジェクト名が「a-ishow-test」の場合、以下のようなURLが使用されます：

- **本番環境**: `https://a-ishow-test.vercel.app`
- **または**: `https://a-ishow-test-koywsmz97-gmailcoms-projects.vercel.app`

## 各ページのリンク

プロジェクトのメインURLが `https://あなたのプロジェクト名.vercel.app` の場合：

### 参加者用リンク
```
https://あなたのプロジェクト名.vercel.app/
```

### 管理者用リンク
```
https://あなたのプロジェクト名.vercel.app/admin
```

### 大画面モニター用リンク
```
https://あなたのプロジェクト名.vercel.app/display/イベントID
```
※イベントIDは管理者ページで確認できます

## 確認事項

- ✅ サイトが正常に表示されるか
- ✅ 参加者登録ができるか
- ✅ 管理者ページにアクセスできるか
- ✅ マッチング閾値の設定が正しく動作するか
